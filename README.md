# Angular-CRUD-Full-Stack-Version

This is a sample **Angular Fullstack CRUD** application demonstrating a modern Angular 16+ standalone setup with a .NET Web API backend. It supports **CRUD operations**, uses **Swagger** for API testing, and is configured for **production deployment** to GitHub Pages with a live Azure API.

## Features

- Full CRUD operations for users
- Angular 16+ standalone components
- .NET Web API backend with Entity Framework Core
- Swagger documentation in development
- Configured CORS for GitHub Pages
- Ready for production deployment

---

## Tech Stack

- **Frontend:** Angular 16+, TypeScript, SCSS
- **Backend:** .NET 8 Web API, Entity Framework Core, SQL Server
- **Hosting:** GitHub Pages (frontend), Azure App Service (API)
- **Other Tools:** Swagger, Angular CLI, `angular-cli-ghpages`

---

## Prerequisites

- Node.js >= 20
- npm >= 9
- Angular CLI >= 16
- .NET 8 SDK
- Git
- Azure subscription (for API deployment)
